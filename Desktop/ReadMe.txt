Dictionary
Made by Trevor Bird

******************************************

What does it do:
This program takes a .txt file and looks for any words in the file and puts it in a dictionary that prints at the end of the program, listing all different words and how many times that word appears.

******************************************

How To Install:
1. Open terminal.
2. Use the cd command to get to the directory location of the files
3. Type "make" without the quotes.

******************************************

How To Run:
1. Open terminal.
2. Use the cd command to get to the directory location of the .exe file.
3. Type "Dictionary < data.txt" without the quotes.

Note: replace data.txt with the name of the file you are using to generate a dictionary from if it is not called data.txt.
